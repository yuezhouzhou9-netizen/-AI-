# 运宝小七正文配图 Skill 使用说明

`my-cyber-illustrations` 是一个面向中文内容创作的配图 Skill。它会先理解文章中的关键观点、流程、状态和隐喻，再将其转化为风格统一的 16:9 横版正文插图。

仓库地址：[yuezhouzhou9-netizen/-AI-](https://github.com/yuezhouzhou9-netizen/-AI-)

## 它能做什么

这个 Skill 不只是“根据文字生成一张插画”，而是帮助你完成一套正文配图工作流：

- 分析文章中哪些段落最值得配图
- 输出配图规划（shot list）
- 为每个关键观点生成独立插图
- 保持固定 IP 角色、色彩和构图风格一致
- 修复图片中的常见问题，例如错位、错误标题或角色特征缺失

默认主角是“运宝小七”：一只穿白色太空战衣、戴星星天线头盔、胸前带“7”徽章的 Q 版赛博小猪。

## 适合哪些内容

尤其适合以下场景：

- 中文公众号、博客、Notion 文档的正文配图
- AI 工作流、方法论、知识管理和效率工具类内容
- 对比、流程、状态变化、系统结构等抽象概念
- 希望建立稳定个人视觉识别的内容账号

它不适合用于复杂架构图、传统 PPT 信息图、商业海报、品牌 KV，或需要严格可编辑矢量文件的场景。

## 输出风格

每张图遵循以下视觉规范：

- 16:9 横版
- 纯白背景与充足留白
- 圆润、干净的 3D 科技吉祥物风格
- 主角必须参与核心动作，而不是站在旁边装饰
- 少量蓝、橙、红、紫色中文手写批注
- 主体整体居中，避免一边拥挤、另一边空白
- 一张图只表达一个关键结构或认知动作

## 如何使用

安装完成后，在支持 Skill 调用的 Agent 中使用类似提示词：

```text
Use $my-cyber-illustrations 为下面这篇中文文章设计并生成正文配图。

要求：16:9 横版、纯白背景、运宝小七 IP、
少量蓝橙红紫中文手写批注。

<粘贴文章内容>
```

### 1. 只做配图规划

如果你还不想立即生成图片，可以先让它输出 shot list：

```text
Use $my-cyber-illustrations 先不要生图。

请分析下面文章哪些位置适合配图，
输出 5 张左右的 shot list。
每张图写清楚：放在哪段后、主题、核心意思、
结构类型、主角动作和建议中文标注词。

<粘贴文章内容>
```

### 2. 直接为文章生成多张配图

```text
Use $my-cyber-illustrations 把下面这篇文章生成 4 张正文配图。

<粘贴文章内容>
```

通常一篇文章建议生成 4–8 张图。短内容可以只做 1–3 张；不必为每个段落平均配图，应优先选择真正承担“认知转折”的部分。

### 3. 为一个观点单独生成插图

```text
Use $my-cyber-illustrations 为“知识不是读来的，
而是一块一块积累进去的”生成一张正文配图。

主角必须承担核心动作，画面简洁、白底、
有科技感和一点俏皮感。
```

### 4. 修改已有图片

它也可以用于修图。例如删除图片里多余的标题：

```text
Use $my-cyber-illustrations 帮我编辑这张图，
去掉左上角的“流程图”标题，其他内容保持不变。
```

也可以要求修复主角缺失的星星天线头盔、调整主体居中，或减少图片中错误、过多的中文标注。

## 使用建议

- 一张图只讲一个重点，不要把整篇文章压缩成说明书。
- 图片中的中文越短越稳定，建议每处控制在 2–8 个字。
- 优先把抽象概念转成具体动作，例如“筛选信息”“搬运数据块”“打开黑盒”“拉住流程线”。
- 生成后检查主角是否仍具备猪鼻、猪耳、蓝色面罩、星星天线头盔、“7”徽章和蓝紫披风。
- 如果连续两次生成仍不理想，建议换一种隐喻或构图，而不是持续微调同一条提示词。

## 最终效果

理想的成图应该让读者一眼感到：这不是普通装饰插画，而是用“运宝小七正在完成一项星际任务”的方式，帮助自己快速理解文章中的一个关键想法。
